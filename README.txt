Team project - Big Data Analytics Class

Dataset : 
Previous grades and features about each students.
More details on the features in the “student.txt” file.
Available on https://archive.ics.uci.edu/ml/datasets/student+performance

The goal of this project is to predict student’s performance given a set of features.
The final objective being to be able to produce a tool for teachers, to know which students are more likely to need particular support to pass.
